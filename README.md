# js-fixed-header

Create fixed header in your table. Write in Vanilla JS.

## Installation

### Bower
<pre>
bower install js-fixed-header
</pre>

## How to use 

### Create
```javascript
var fixedTable = new JSFixedHeader(table,options);
```

### Destroy
```javascript
fixedTable.destroy();
```

### Default options 
```javascript
{
    "top" : 0,
    "zIndex" : 100
}
```
